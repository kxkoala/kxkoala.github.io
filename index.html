1.常见运算符及其表达式

1.1算术运算符

加+ 减- 乘* 除/ 求

函数名 功能

sqrt(x) 平方根
log(x) ln(x)

pow(x) x的y次方
div(int x,int y) 整数除法的商和余数

开头加#include <math.h>

1.2赋值运算符

作用：将表达式的值赋给一个左值

形式：左值 = 表达式

运算符 运算实例 展开形式
+= a+=b a=a+b
-= a-=b a=a-b
*= a*=b a=a*b
/= a/=b a=a/b
%= a%=b a=a%b

优先级：高于逗号运算符，低于算数运算符、关系运算符、逻辑运算符

1.3自增自减运算符

++/– 给变量的值增加/减少1

例：（x=5)

实例 等价于 表达式 表达式的值 表达式后x的值
y=x++ y=x,x=x+1 5+x++ 10 6
y=++x y=x+1,x=x+1 ++x+5 11 6
y=–x y=x-1,x=x-1 –x+5 9 4

​ 后缀++/–优先级高于前缀++/–及其他单目运算符

​ （左结合） （右结合）

1.6类型转换运算符

双目运算符两端的运算对象类型不同时，将较低的类型转换为较高的类型，再参加运算。

char,short→int→unsigned→long→double→float

​ 低级 ——————————|——→ 高级

1.6.1自动类型转换

1）数据长度增加

2）字节换成较高

3）有/无符号类型都在，转换为无符号

4）浮点运算都为双精度

5）char和short型参与运算时，先转换成int型

6）赋值运算时，运算符右边量的类型先转换为左边量的类型

1.6.2强制类型转换

形式：（类型标识符）（表达式）

1.7逗号运算符

（优先级最低）

形式：表达式1，表达式2，表达式3....表达式n

整个逗号表达式的值时表达式n的值

2.选择结构程序设计

2.1 if语句

if（条件）{语句1；} else{语句2；}

或if（条件）{语句1；}

2.2条件表达式的值

含义为真或假，0为假，非0为真

赋值表达式的值为表达式中变量的值，而等于运算符的值只能是真或假（1或0）

2.3条件运算符

1)条件运算符为‘ ? ’和‘ ：’，是三目运算符

2)形式：表达式1?表达式2：表达式3

3)求值规则：若表达式1 为真，则以表达式2 的值作为条件表达式的值；否则以表达式3作为条件表达式的值。

4)通常用于：赋值语句

​ 例：if(a>b)max=a;

​ else max=b

​ 可写为 max=((a>b)?a:b);

5)优先级：低于关系运算符和算术运算符,所以 4）第二种写法的所有括号都可以省略

6)结合方向：自右至左

2.4选择结构的嵌套

例：

#include<stdio.h> int main() { int year, x; printf("输入年份："); scanf("%d", &year); if (year % 4 == 0 && year % 100 != 0) x = 1; else if (year % 400 == 0) x = 1; else x = 0; printf("%d", x); return 0; }

2.5 switch分支选择

例：

#define _CRT_SECURE_NO_WARNINGS #include<stdio.h> int main() { int x=0, a=x/10; char A, B, C, D, E; scanf("%d,%d,%c,%c,%c,%c,%c", &x,&a,&A,&B,&C,&D,&E); a = x / 10; switch (a) { case 0:case 1:case 2:case 3:case 4:case 5:printf("E"); break; case 6:printf("D"); break; case 7:printf("C"); break; case 8:printf("B"); break; case 9:case 10:printf("A"); break; default:printf("Input error!"); } return 0; }

（break：跳出当前的结构）
